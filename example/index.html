react-colorpickr

A colorpicker for React. Fork it on GitHub.

Install

npm install react-colorpickr

You'll also want to include a copy of react-colorpickr.css in your code.

Usage

import React from 'react'
import ColorPicker from 'react-colorpickr'

class App extends React.PureComponent {
  onChange = color => {
    console.log(color);
  };

  render() {
    return <ColorPicker onChange={this.onChange} />
  }
}
