react-colorpickr

A colorpicker for React. Fork it on GitHub.

Install

npm install @mapbox/react-colorpickr

You'll also want to include a copy of react-colorpickr.css in your code.

Usage

import React from 'react'
import ColorPicker from '@mapbox/react-colorpickr'

function App() {
  return <ColorPicker onChange={console.log} />
}
